# zufunkt

Sammelrepo für von agentX generierte Dateien.

## Ablageregel

Format:
- `images/YYYY-MM-DD/<projekt>/`
- `video/YYYY-MM-DD/<projekt>/`
- `audio/YYYY-MM-DD/<projekt>/`
- `docs/YYYY-MM-DD/<projekt>/`
- `misc/YYYY-MM-DD/<projekt>/`

Beispiel:
- `images/2026-04-14/avatar-konzept/`

## Bereiche
- `images/`
- `video/`
- `audio/`
- `docs/`
- `misc/`
